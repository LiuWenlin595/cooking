# 图片资源说明

本目录用于存放小程序所需的图片资源。

## 目录结构

- `tabbar/` - 底部导航栏图标
  - `home.png` / `home-active.png` - 首页图标
  - `recipe.png` / `recipe-active.png` - 菜谱图标
  - `order.png` / `order-active.png` - 订单图标
  - `profile.png` / `profile-active.png` - 我的图标

- `default-avatar.png` - 默认头像
- `default-bg.jpg` - 默认背景图
- `default-recipe.png` - 默认菜谱图片

## 注意事项

1. 所有图片建议使用 PNG 格式（除背景图可使用 JPG）
2. 图标尺寸建议：81x81px（@2x）或 162x162px（@3x）
3. 菜谱图片建议尺寸：750x500px
4. 头像和背景图可根据实际需求调整尺寸

## 占位图片

如果暂时没有图片资源，可以使用以下方式：
1. 使用在线占位图服务
2. 使用设计工具生成简单占位图
3. 在小程序中，未设置图片时会显示默认占位图
