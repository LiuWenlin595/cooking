# 快速开始指南

## 🚀 5分钟快速上手

### 第一步：打开项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `CookingApp` 文件夹
4. 填写 AppID（或选择"测试号"）
5. 点击"确定"

### 第二步：准备图片（可选）

如果暂时没有图片，可以：
- 跳过此步骤，小程序会使用默认占位
- 或使用在线占位图服务生成临时图片

### 第三步：运行项目

1. 点击"编译"按钮
2. 在模拟器中查看效果
3. 点击"预览"在手机上测试

### 第四步：开始使用

1. **设置店铺**：进入"我的" → "店铺管理"
2. **添加菜谱**：进入"菜谱" → "添加菜谱"
3. **开始点菜**：浏览菜谱并下单

## 📱 功能快速体验

### 体验流程

1. **首页浏览**
   - 查看店铺信息
   - 浏览推荐菜谱
   - 查看最近订单

2. **添加菜谱**
   - 进入"菜谱"页面
   - 点击"添加菜谱"
   - 填写基本信息、用料、步骤
   - 保存

3. **点菜下单**
   - 浏览菜谱列表
   - 点击查看详情
   - 点击"点这道菜"
   - 确认下单

4. **管理订单**
   - 进入"订单"页面
   - 查看订单列表
   - 管理员可以接单和出餐

5. **食材管理**
   - 进入"食材管理"
   - 添加现有食材
   - 查看可做菜谱

6. **饮食计划**
   - 进入"饮食计划"
   - 选择日期
   - 安排每日菜单
   - 一键下单

## ❓ 常见问题

**Q: 图片不显示怎么办？**
A: 检查图片路径，或使用在线图片URL

**Q: 数据丢失了？**
A: 使用"导出数据"功能定期备份

**Q: 如何切换厨房？**
A: 进入"店铺管理" → 点击厨房的"切换"按钮

**Q: 如何成为管理员？**
A: 在"店铺管理"中，由现有管理员添加你为管理员
